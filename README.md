# Prácticas — Medicina

Quizzes bilingües (ES/EN) de inmunología y dermatología basados en las clases de la Dra. Andrea Ramírez Fontes.

- 9 temas · 20 preguntas por tema · 180 preguntas en total
- Modo **por tema** o **aleatorio** (mezcla 25 preguntas)
- Next.js 14 · TypeScript · Tailwind · Framer Motion

## Desarrollo local

```bash
npm install
npm run dev
```

Abre http://localhost:3000.

## Despliegue

Importa este repo en [vercel.com](https://vercel.com) — Next.js se autodetecta sin configuración.
